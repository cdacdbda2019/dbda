Note on K-Means++ for initilization of centroids¶

  • For example, We need to select 2 cluster centers.

  • Select the first centroid randomly,

  • Find the points that are farthest to the first center {These points most probably do not belong to the first cluster center as they are far from it}.

  • Assign the second cluster center nearby those far points.

Example:¶

Our observations are [0, 1, 2, 3, 4].

Let the first center, c1, be 0.

p(c2): The probability that the next cluster center, c2, is x is proportional to ||c1-x||^2.

  • P(c2 = 1) = 1w, P(c2 = 2) = 4w, P(c2 = 3) = 9w, P(c2 = 4) = 16w, where w = 1/(1+4+9+16).

p(c2=1) = 1/30

p(c2=2) = 2/30

p(c2=3) = 3/30

p(c2=4) = 4/30 --> Which hisgest probability, pick it as c2

So, c2=4.
